# NeuralSite 核心方向：文档驱动的3D可视化系统

**记录时间：** 2026-03-06

---

## 核心理念

```
图纸信息 → 文档/配置(JSON) → 3D模型自动变化

死图纸 → 变活
```

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 分段施工 | 挡墙分多段（每50米一段） |
| 分层施工 | 每段分基础层、墙身层、帽石层 |
| 状态显示 | 未建(虚线) → 施工中(橙色) → 已建(绿色) |
| 4D时间 | 按进度时间动画（从下往上生长） |
| 数据准确 | 工程必须精确 |

---

## JSON文档结构

```json
{
  "project": { "name": "挡墙项目", "total_length": 150 },
  "visual_rules": {
    "未建": { "color": "#A0A0A0", "line_style": "dashed" },
    "施工中": { "color": "#FF9800", "line_style": "solid" },
    "已建": { "color": "#4CAF50", "line_style": "solid" }
  },
  "segments": [
    {
      "id": "seg_001",
      "layers": [
        { "name": "基础层", "status": "已建", "progress": 100 },
        { "name": "墙身层", "status": "施工中", "progress": 65 },
        { "name": "帽石层", "status": "未建", "progress": 0 }
      ]
    }
  ]
}
```

---

## 约定俗成

| 表达 | 含义 |
|------|------|
| 虚线 | 未建 |
| 实线 | 已建/施工中 |
| 颜色灰 | 未建 |
| 颜色橙 | 施工中 |
| 颜色绿 | 已建 |
